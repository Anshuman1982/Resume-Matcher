# Resume Matcher 

A Streamlit app that matches resumes to job descriptions based on skill similarity.

## Features
- Upload resume (PDF)
- Upload job description (text)
- See matched skills and % similarity
- Clean UI and instant results

##  Tech Stack
- Python
- Streamlit
- PyPDF2, sklearn, nltk
